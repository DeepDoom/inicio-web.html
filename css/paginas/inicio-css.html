Css

En esta pagina veremos todo lo que tiene que ver con Css
(Cascading Stile sheets)

Las hojas de Css, básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML y incluyendo los lenguajes basados en XML, que se encarga del contenido básico de las páginas.

Primero para empezar a programar nesecitamos el programa sublime text, es la plataforma que usaremos para escribir nuestro codigo. Debemos crear en el sublime text un archivo nuevo "File-New File" y en guardarlo "File-Save As", y lo guardamos con el nombre de "Css para mi primera.css" (si alfinal del nombre del archivo no ponemos .css no nos dejara programar y no reconocera los codigos).

Ya teniendo nuestro archivo creado y guardado tendremos que hacer que nuestro archivo css este comuniado con nuestro html para poder ir viendo nuestro avance, mediante el sigiente elemento de HTML:

< link rel="stylessheet" type="text/css" href="Css para mi primera web.css">

A diferencia de HTML, Css no posee ni head ni body, todo el contenido es agregado mediante uon elementos o clases agregados en html.

Anatomía de una regla CSS:

La estructura completa es llamada "regla predeterminada" (pero a menudo < reglra >):

  • Selector: El elemento HTML en el que comienza la regla. Esta selecciona el/los elementos a dar estilo, (en este caso, los elementos < p >). Para dar estilo a un elemento diferente, solo cambia el sector.
  • Declaración:Una sola regla como color:red; especifica a cuál de las propiedades del elemento quieres dar estilo.
  • Propiedades:Maneras en las cuales puedes dar estilo a un elemento HTML. (En este caso, color es una propiedad del elemento < p >). en CSS, seleccionas qué propiedad quieres afectar en tu regla.
  • Valor de la prioridad:A la derecha de la propiedad, después de los dos puntos (:), tienes el valor de la propiedad, para elegir una de las muchas posibles apariencias para una propiedad determinada (hay muchos valores para color además de red).
