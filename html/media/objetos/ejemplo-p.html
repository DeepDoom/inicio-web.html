html

HTML se utiliza para definir una estructura básica y un código para la definicion de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Html se considera el lenguaje web más importante siendo su invención creucial en la aparicion, desarrollo y expansión de la Word Wide Web (WWW). Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales utilizan.
