Que es <dl>

El elemento <dl> representa una lista desciptiva. El elemento encierra una lista de grupos de términos especificados con el uso del elemento <dl> y de descripciones proveidas con elementos <dd>. Algunos usos comunes para este elementos son implementar un glosario o para desplegar metadatos lista de pares llave-valor.

Uso:

El elemento <dl> no debe ser utilizado solo para crear sangria en una página, Si bien funciona, es una mala práctica y obscurece el significado de las listas descriptivas.

HTML permite encapsular cada grupo nombre-valor de un elemento <dl> en un elemento <div>. Esto puede ser útil cuando se utiliza microdatos, o cuando atributos globales apliquen a todo el grupo, o por motivos de estilos.

atributos:

Este elemento sólo inlcuye atributos globales.
