Elemento HTML
(<dd>)

Que es <dd>

El elemento <dd> provee detalles acerca de o la definición de un término precedente <dt> en una lista de descripciones.

Uso:

El elemento <dd> se utiliza adentro de un elemento <dl> y acompañando a un elemento <dt> para darle mas informacion.

atributos:

Este elemento sólo inlcuye atributos globales.
