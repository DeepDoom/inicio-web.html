Que es <address>

El elemento HTML <address> aporta información para su <article> más cercano o ancentro <body>; en el último caso lo aplica a todo el documento.

Uso:

Para representar una dirección arbitraria, una que no esté relacionada con la información de contacto, utiliza un elemento <p> en lugar del elemento <address>.

Este elemento no debe contener más información que la información de contacto, como una fecha de publicación que pertenece al elemento <time>.

Normalmente un elemento <address> puede ser colocado dentro del elemento <footer> de la sección actual, si lo hay.

Contenido permitido

Contenido dinámico, pero No con elementos address anidados, No contenidos de cabecera {hgroup, h1-h6}, No contenido de sección {article, aside, section, nav}, y No elemento header o footer

atributos:

Este elemento sólo inlcuye atributos globales.
