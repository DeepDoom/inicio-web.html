Que es <q>

El elemento <q> indica que el texto adjunto es una cita corta en linea. La mayoria de los nacefadores modernos implementan esto redeando el texto entre comillas. Este elemento está destinado a citas breves que no requieren saltos de párrafo; para citas de bloque independiente, utiliza el elemento <blockquote>.

atributos:

Este elemento sólo inlcuye atributos globales.

Atributos Globales

cite: El valor de este atributo es una URL que desufna un documento o mensaje fuente para la información citada. Este atributo está destinado a señalar indormación que explica el contexto o la referencia de la cita.
