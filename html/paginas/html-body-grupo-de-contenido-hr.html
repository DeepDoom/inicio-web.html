Elemento HTML

Que es <hr>

El elemento <hr> representa una cambio de tema entre párrafos (por ejemplo, un cambio de ecena en una historia, un cambio de tema en una sección). En versiones previas de HTML representaba una linea horizontal. Aún puede ser reoresentada como una lunea horizontal un los navegadores visuales, pero ahora es definida en términos semanticos y no tanto en términos representativos, por tanto para dibujar una linea horizontal se deberia de usar el CSS apropiado.

Uso:

El elemento <hr> no puede contener Nada, es un "elemento vacio".

atributos:

Atributo Descripción Valor
title Titulo consultivo del elemento. Texto legible por personas.
style Información de estilo en linea. Reglas de estilo.
id Identificador único a nivel de documento. ID.
class Identificador a nivel de documento. Lista de clases separadas por espacios.
dir Dirección del texto débil/neutral. Uno de los siguientes: "ltr" o "rtl".
lang Información sobre el idioma. Código de idioma.
aling Indica la alineación horizontal con relación al contexto. Uno de los siguientes: "left" "center", "right" o "justify".
noshade Elimina el relieve. "noshade".
size Indica la altura de la linea. Una cantidad, de pixeles.
width Indica la anchura. Una cantidad, de pixeles, o un porcentaje.
