Elemento HTML
(<style>)

Que es <style>

El elemento <style> de HTML es encargado de inficar la información de estilo.

Uso:

El elemento <style> debe ser contenido por el Elemento <head>.

atributos:

Atributo Descripción Valor
title Texto informativo o titulo del elemento, suele mostrarse a modo de "tool tip". Texto legible por personas. Sensible a M/m. Por defecto: Lo fija el navegador.
lang Información sobre el idioma del contenido del elemento y del valor de sus atributos. Un codigo de idioma. Por defectos: "desconocido"- Lo fija el navegador.
dir Indica la dirección de texto y tablas. Uno de los siguientes: "ltr" o "rtl". Por defecto: En castellano "ltr". Lo fija el navegador.
type Indica el lenguaje de hojas de estilo usado. Un tipo MIME. Para CSS es:"text/css".
media Indica el/los medios a los que estpa destinada la información de estilo. Uno o más descriptores de medio separados por coma. Por defecto: "screen". Lo fija el navegador.

Ejemplos:

Código
    Style Inline
  1. <html>

      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title> </title>
    </head>

      <p style= "color: red;">Ejemplo de estilos</p>
    </body>
    <html>

    </head>
    <body>
      <p>Ejemplo de estilos</p>
    </body>
