Que es <figcaption>

El elemento <figcaption> representa un subtitulo o leyenda asociado al contenido del elemento padre <figure>, pudiendo ser colocado como primer o último hijo. Es importante destacar que el elemento <figucaption> es opcional.

Uso:

El elemento <figcaption> se utiliza para acompañar al elemento <figure>.

atributos:

Este elemento sólo inlcuye atributos globales.
