Elemento HTML

Que es <nav>

El elemento HTML <nav> representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e indices.

Uso:

No es necesario que todos los enlaces estén contenidos en un elemento <nav>. <nav> es´ta destinado sólo para el bloque principal de enlaces de navegación; por lo general, el elemento <footer> a menudo contiene una lista de enlaces que no necesitan estar en un elemento <nav>.

Un documento puede tener varios elementos <nav>, Por ejemeplo, uno para la navegación del sitio y otra para la navegación dentro de la página. aria-labelledby puede ser utilizado en tal caso para promover la accesibilidad.

Los agentes de usuario, como los lectores de pantalla dirigidos a usuarios con discapacidades, pueden usar este elemento para determinar si se omite la representación inicial del contenido de solo navegación.

atributos:

Este elemento sólo inlcuye atributos globales.
