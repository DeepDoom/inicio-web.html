El lenguaje de mercado de hipertexto HTML
(Hipertext Markup Language)

Que es HTML:

HTML se utiliza para definir una estructura básica y un código para la definicion de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Html se considera el lenguaje web más importante siendo su invención creucial en la aparicion, desarrollo y expansión de la Word Wide Web (WWW). Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales utilizan.

Primeros pasos:

Primero para empezar a programar nesecitamos el programa sublime text, es la plataforma que usaremos para escribir nuestro codigo. Debemos crear en el sublime text un archivo nuevo "File-New File" y en guardarlo "File-Save As", y lo guardamos con el nombre de "inicio.html" este archivo lo podremos abrir mediante el navegador para poder ir viendo el avance de nuetro trabajo (si alfinal del nombre del archivo no ponemos .html no nos dejara programar y no reconocera los codigos).

Ya teniendo nuestro archivo creado y guardado empezaremos con los codigos base, el primer codigo que pondremos es el: < html > el sublime text cuando escribimos nos da la opcion de auto completar, al escribir html no saldra una opcion tocando esa opcion no creara toda la base de nuestra pagina.

< head >< /head > El elemento HTML < head > provee informacion general (metadatos) acerca del documento, incluyendo su titulo y enlaces a scripts y hojas de estilos.. es la parte que no se muestra en el navegador en el momento de cargar la pagina, Esta etiqueta puede contener varios codigos que otorgan información sobre el sitio en cuestion.

Secciones

Las secciones son las partes temáticas en las que se puede divifir el contenido de una página web.

En HTML 4 / XHTML 1 la única etiqueta para definir agrupaciones de contenido era prácticamente la etiqueta < div >. por ello, cualquier página de tamaño medio acababa teniendo muchos elementos < div >, en algunos casos para organizar el contenido de la página. pero también para aplicar estilos de carpacter decorativo. El código fuente resultaba asi dificil de enterder y mantener.

< section > < nav > < article > < aside > < h1,- h6 > < header > < footer > < address >

Grupo de contenido

Los los elementos que podemos usar para agrupar contenido y organizar informacion.

como por ejemplo crear párrafos o listas.

< p > < hr > < pre > < blockquote > < ol > < ul > < li > < dl > < dt > < dd > < figure > < figcaption > < div >

Semantica, nivel de texto

La semántica en el HTML y HTML5 se refiere al significado o información extra que otorgan los elementos o etiquetas del lenguaje, información que define o describe el contenido, función o sección que contiene.

Esta información es lo que permite que un documeto se pueda rastrear de mejor manera.

< a > < abbr > < acronym > < b > < bdo > < big > < br > < cite > < code > < dfn > < em > < font > < i > < kbd > < mark > < q > < samp > < small > < span > < strike > < strong > < sub, sup > < time > < tt > < u > < var > < wbr >

Ediciones

Ediciones de texto, se recomienda hacer estos cambios con Css y no con HTML.

< ins > < del >

Contenido incrustado

El contenido incrustado se utiliza para mostrar recursos externos como, por ejemplo, mapas, videos, audios y imagenes entre otros.

Este metodo permite utilizar un servicio o interfaz de terceros.

< img > < iframe > < embed > < object > < video > < audio > < track > < canvas > < map > < area > < svg > < frame > < frameset > < noframes >

Tablas

Una tabla HTML es un conjunto de celdas organizadas en filas que a su vez se organizan en grupos de filas.

Ademas, la tabla puede tener una leyenda y hacer referencia a las columnas.

< table > < caption > < colgroup > < col > < tr > < td > < th >

Formularios

Un formulario HTML es una sección elementos especiales llamados "controles" check boxes, radio buttons, labes, etc..

Los usuarios normalmente completan un formulario modificando sus parametros antes de enviar el formulario a un agente para que lo precese un servidor. .

< from > < legend > < label > < input > < button > < select > < datalist > < optgroup > < option > < textarea > < output > < progress > < meter >

Interactivos

Estos elementos sirven para crear una pestaña con datos extras.

< details > < summary >
