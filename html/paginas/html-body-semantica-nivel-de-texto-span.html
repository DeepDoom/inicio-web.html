Que es <span>

El elemento <span> es un contenedro en linea. Sirve para aplicar estilos al texto o agrupar elementos en linea.

atributos:

Atributo Descripción Valor
title Titulo consultivo del elemento. Texto legible por personas.
style Información de estilo en linea. Reglas de estilo.
id Identificador único a nivel de documento. ID.
class Identificador a nivel de documento. lista de clases separadas por espacios.
dir Dirección del texto débil/neutral. Uno de los siguientes: "ltr" o "rtl".
lang Información sobre el idioma. Código de idioma.
datasrc Enlaza a una fuente de datos externa. Reservado para posibles usos futuros. Una dirección URL. Por defecto: lo define el navegador.
datafld El nombre de una propiedad o columna de datasrc. Reservado para posibles usos futuros. Por defecto: lo fija el navegador.
dataformatas Indica como deben ser tratados los datos obtenidos de datasrc. Reservado para posibles usos futuros. Uno de los siguientes: "paintext" o "html". Por defecto: "plaintext".
Eventos: onclik, ondblclick, onmousedown, onmouseup, onmouseover, onmousemove, onmouseout, onkeypress, onkeydown, onkeyup
