Que es <article>

El elemento HTML <article> representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable, por ejemeplo, en la indicación. Podria ser un mensaje en un foro, un articulo de revista o un periódico, una entrada de blog, un comentario de usuario, un widget interactivo o gadget, o cualquier otro elemento independiente del contenido.

Uso:

Cuando los elementos <article> están anidados, los internos representan articulos relacionados con el exterior. Por ejemplo, los comentarios de un blog puede ser elementos <article> anidados al que representa la entrada del blog.

Los datos del autor de un elemento <article> pueden ser proporcionados a través del elemento <address>, pero no se aplica a los elementos <article> anidados.

La fecha de publicación y el tiempo de un elemento <article> pueden ser descritos con el atributo pubdate atributo de un elemento <time>.

atributos:

Este elemento sólo inlcuye atributos globales.
